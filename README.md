# splatag-back 基于Pillow的splatoon3名片生成工具

所需环境：python及Pillow模块；  
使用方式：在splatagProcessor.py的main中，按注释提示修改用户名、背景文件名、徽章文件名、称号、id等参数即可，运行即可；  
使用资源：背景、徽章、称号为解包文件，日文英文字体为SplatoonFontFix，中文字体为文悦新青年；
